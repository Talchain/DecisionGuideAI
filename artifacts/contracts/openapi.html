Server-Sent Events stream for real-time decision analysis

Establishes SSE connection for streaming analysis results. Frozen event types: hello|token|cost|done|cancelled|limited|error

Authorizations:
ApiKeyAuth
query Parameters
sessionId
required
string <uuid>

Unique session identifier

analysisType
string
Default: "decision"
Enum: "decision" "scenario" "impact"

Type of analysis to perform

Responses

Development server

http://localhost:3001/stream

Staging server

https://api-staging.decisionguide.ai/stream

Production server

https://api.decisionguide.ai/stream

  • 400
  • 429
  • 500
