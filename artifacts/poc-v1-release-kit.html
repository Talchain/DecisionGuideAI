⚠️ What Can Go Wrong - Quick Fixes

🔧 Common Issues & Solutions

No Output / Blank Screen

The fixture isn't loading or JavaScript is disabled

Quick Fix: Refresh page, check browser console (F12), ensure JavaScript enabled
Hard refresh: Ctrl+Shift+R (Windows) / Cmd+Shift+R (Mac)

Duplicate Tokens Appearing

Stream resume fixture showing token duplication

Quick Fix: This is intentional - demonstrates deduplication feature
Expected: Tokens repeat, then resume from last-event-id

Empty Report Drawer

Report section shows "No data available"

Quick Fix: Use the report-no-data fixture - this is the expected demo state
Navigate: ui-fixtures/report-no-data/view-model.json

Stream Won't Cancel

Cancellation button not working in demo

Quick Fix: Switch to stream-cancel-mid fixture which shows pre-cancelled state
Fixture shows: Stream stopped with reason: "cancelled"

Jobs Progress Stuck

Progress bar not updating in jobs demo

Quick Fix: Use jobs-cancel-50 fixture - shows progress 0→50→cancelled
Expected: 10 steps progress, cancelled at 50%

Error Stream Too Scary

Error message looks too technical for stakeholders

Quick Fix: Emphasize the user-friendly error handling in stream-error fixture
Show: "Analysis temporarily unavailable, please try again" message
